Операційні системи GNU/Linux

Коротка інформація про GNU/Linux

GNU/Linux – це операційна система, заснована на ядрі Linux та інструментах GNU, що відома своєю відкритою ліцензією і високою гнучкістю. Вона є популярною серед розробників, системних адміністраторів і користувачів, які цінують стабільність, безпеку та свободу налаштування. GNU/Linux має багато дистрибутивів (версій), які підходять для різних потреб.

Найпопулярніші дистрибутиви

  • Ubuntu: дружній до користувача, популярний для настільних комп'ютерів і серверів.
  • Debian: стабільний і безпечний, на основі якого створено багато інших дистрибутивів.
  • Fedora: призначений для розробників, має актуальні версії програмного забезпечення.
  • openSUSE: підходить для серверів та настільних систем, має потужні інструменти адміністрування.
  • Arch Linux: призначений для досвідчених користувачів, дозволяє повністю налаштувати систему.
  • CentOS: стабільний і надійний для серверів, базується на Red Hat Enterprise Linux.

Приклади команд Linux CLI

  • pwd — показує поточний каталог
  • echo — виводить повідомлення
  • ls — список файлів у каталозі
  • cd [каталог] — змінює поточний каталог
  • touch [ім'я файлу] — створює новий порожній файл
  • mkdir [ім'я каталогу] — створює новий каталог
  • cp [джерело] [призначення] — копіює файли або каталоги
  • mv [джерело] [призначення] — переміщує або перейменовує файли чи каталоги
  • rm [ім'я файлу або каталогу] — видаляє файли або каталоги
  • history — показує список останніх використаних команд

BASH-скрипт для імітації авторизації


#!/bin/bash
echo "Ласкаво просимо до системи!"
read -p "Введіть ім'я користувача: " username
read -sp "Введіть пароль: " password
echo

# Задаємо логін і пароль
correct_username="admin"
correct_password="password123"

# Перевірка логіна та пароля
if [[ "$username" == "$correct_username" && "$password" == "$correct_password" ]]; then
    echo "Авторизація успішна. Ласкаво просимо, $username!"
else
    echo "Неправильний логін або пароль. Доступ заборонено."
fi
